Project Overview
A comprehensive machine learning solution for predicting diabetes risk using patient medical data. This project implements and
compares multiple classification algorithms to identify the most effective model for diabetes prediction.


Features
Data Preprocessing: Cleans and prepares diabetes dataset by handling missing values and outliers
Multiple Algorithms: Implements 4 different ML algorithms for comprehensive comparison
Visual Analytics: Generates insightful visualizations including correlation heatmaps and performance charts
Model Evaluation: Uses multiple metrics (Accuracy, Recall, F1-Score, ROC-AUC) for thorough assessment
Result Export: Automatically saves model performance results to CSV format

Algorithms Implemented
Logistic Regression
Random Forest
Gradient Boosting
Support Vector Machine

Performance Metrics
Accuracy: Overall correctness of predictions
Recall: Ability to identify diabetic cases correctly
F1-Score: Harmonic mean of precision and recall
Confusion Matrix: Detailed breakdown of predictions
ROC-AUC: Area under the ROC curve for classification quality


Visualizations
Correlation Heatmap: Shows relationships between features
Performance Bar Charts: Individual metric comparisons
Combined Line Chart: All metrics comparison across algorithms
Confusion Matrices: Detailed prediction analysis for each model
